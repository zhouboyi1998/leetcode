### ❓ Description

<br/>

给你一个整数数组 `nums`，有一个大小为 `k` 的滑动窗口从数组的最左侧移动到数组的最右侧。

你只可以看到在滑动窗口内的 `k` 个数字。

滑动窗口每次只向右移动一位。

<br/>

返回 *滑动窗口中的最大值* 。

<br/>

**示例 1：**

```
输入: nums = [1, 3, -1, -3, 5, 3, 6, 7], k = 3
输出: [3, 3, 5, 5, 6, 7]
解释: 
滑动窗口的位置                 最大值
---------------               -----
[1  3  -1] -3  5  3  6  7       3
 1 [3  -1  -3] 5  3  6  7       3
 1  3 [-1  -3  5] 3  6  7       5
 1  3  -1 [-3  5  3] 6  7       5
 1  3  -1  -3 [5  3  6] 7       6
 1  3  -1  -3  5 [3  6  7]      7
```

**示例 2：**

```
输入: nums = [1], k = 1
输出: [1]
```

<br/>

**提示：**

* `1 <= nums.length <= 10^5`
* `-10^4 <= nums[i] <= 10^4`
* `1 <= k <= nums.length`

---

### ❗ Solution（单调队列）

<br/>

#### idea

* 利用 **双向队列** 队头队尾都可以出队的性质，解决当前 **滑动窗口** 问题
* **队尾** 负责处理数组元素的大小比较
* **队头** 负责移除已经不在窗口内的元素
* *注意：队列中需要存储数组下标，而不是元素本身，因为需要利用下标判断元素是否还在窗口内*

<br/>

* 遍历数组，将 **当前元素** 和 **队尾** 元素进行比较，从 **队尾** 移除队列中所有比 **当前元素** 小的元素
* 移除完后，将 **当前元素** 从 **队尾** 入队

<br/>

* 从 **队头** 开始遍历队列，根据下标判断元素是否还在窗口内，移除所有已经不在 **当前窗口** 内的元素
* 移除完后，**队头** 元素就是 **当前窗口** 的最大值

<br/>

#### Java

```
class Solution {
    public int[] maxSlidingWindow(int[] nums, int k) {
        int n = nums.length;

        // 单调队列 (存储)
        Deque<Integer> deque = new LinkedList<Integer>();

        // 预处理第一个窗口, 将第一个窗口的最大值添加到队列中
        for (int i = 0; i < k; i++) {
            while (!deque.isEmpty() && nums[i] >= nums[deque.peekLast()]) {
                deque.pollLast();
            }
            deque.offerLast(i);
        }

        // 存储滑动窗口中每个窗口的最大值
        int[] result = new int[n - k + 1];
        result[0] = nums[deque.peekFirst()];

        // 遍历处理剩余的窗口
        for (int i = k; i < n; i++) {
            while (!deque.isEmpty() && nums[i] >= nums[deque.peekLast()]) {
                deque.pollLast();
            }
            deque.offerLast(i);

            // 从队头移除在当前窗口之前的元素下标
            while (deque.peekFirst() <= i - k) {
                deque.pollFirst();
            }
            result[i - k + 1] = nums[deque.peekFirst()];
        }

        return result;
    }
}
```

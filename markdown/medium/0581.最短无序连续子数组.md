### ❓ Description

<br/>

给你一个整数数组 `nums`，你需要找出一个 **连续子数组** ，如果对这个子数组进行升序排序，那么整个数组都会变为升序排序。

请你找出符合题意的 **最短** 子数组，并输出它的长度。

<br/>

**示例 1：**

```
输入: nums = [2, 6, 4, 8, 10, 9, 15]
输出: 5
解释: 你只需要对 [6, 4, 8, 10, 9] 进行升序排序, 那么整个表都会变为升序排序
```

**示例 2：**

```
输入: nums = [1, 2, 3, 4]
输出: 0
```

**示例 3：**

```
输入: nums = [1]
输出: 0
```

<br/>

**提示：**

* `1 <= nums.length <= 10^4`
* `-10^5 <= nums[i] <= 10^5`

<br/>

**进阶：** 你可以设计一个时间复杂度为 `O(n)` 的解决方案吗？

---

### ❗ Solution

<br/>

#### idea

* 根据题意，可以将数组分为三段：*左右两段都是有序的，中段无序*
* 需要找到中段无序区间的 **开始下标**、**结束下标**，从而计算出这个无序区间的长度

<br/>

* **从左往右** 遍历
* 如果元素一直升序排列，说明在 **有序区间**
* 如果 *当前元素 `<` 之前遍历的元素中的最大值*，说明进入了 **无序区间**
* 当 *元素再次变为升序有序，且都 `>=` 之前遍历的元素中的最大值*，说明进入了右段的 **有序区间**
* 所以：最后一次出现 *当前元素 `<` 之前遍历的元素中的最大值* 的位置，就是 **无序区间** 的 **结束下标**

<br/>

* **从右往左** 遍历
* 如果元素一直降序排列，说明在 **有序区间**
* 如果 *当前元素 `>` 之前遍历的元素中的最小值*，说明进入了 **无序区间**
* 当 *元素再次变为降序有序，且都 `<=` 之前遍历的元素中的最小值*，说明进入了左段的 **有序区间**
* 所以：最后一次出现 *当前元素 `>` 之前遍历的元素中的最小值* 的位置，就是 **无序区间** 的 **开始下标**

<br/>

* 时间复杂度：`O(n)`
* 空间复杂度：`O(1)`

<br/>

#### Java

```
class Solution {
    public int findUnsortedSubarray(int[] nums) {
        int n = nums.length;

        // 记录从左往右遍历的最大值
        int max = nums[0];
        // 记录从右往左遍历的最小值
        int min = nums[n - 1];

        // 无序区间的开始下标、结束下标
        int begin = -1;
        int end = -1;

        for (int i = 0; i < n; i++) {
            // 从左往右遍历
            if (nums[i] >= max) {
                // 在有序区间, 更新 max 值
                max = nums[i];
            } else {
                // 在无序区间, 更新 end 下标位置 (最后一次更新的 end 下标位置, 就是整个无序区间的 end 下标位置)
                end = i;
            }

            // 从右往左遍历
            if (nums[n - 1 - i] <= min) {
                // 在有序区间, 更新 min 值
                min = nums[n - 1 - i];
            } else {
                // 在无序区间, 更新 begin 下标位置 (最后一次更新的 begin 下标位置, 就是整个无序区间的 begin 下标位置)
                begin = n - 1 - i;
            }
        }

        return end == -1 ? 0 : end - begin + 1;
    }
}
```

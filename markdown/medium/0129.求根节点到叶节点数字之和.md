### ❓ Description

<br/>

给你一个二叉树的根节点 `root`，树中每个节点都存放有一个 `0` 到 `9` 之间的数字。

每条从根节点到叶节点的路径都代表一个数字：

* 例如，从根节点到叶节点的路径 `1 -> 2 -> 3` 表示数字 `123`。

计算从根节点到叶节点生成的 **所有数字之和**。

**叶节点** 是指没有子节点的节点。

<br/>

**示例 1：**

<img src="../../public/0129/num1tree.jpg" alt="num1tree.jpg"/>

```
输入: root = [1, 2, 3]
输出: 25
解释: 
从根到叶子节点路径 1 -> 2 代表数字 12
从根到叶子节点路径 1 -> 3 代表数字 13
因此, 数字总和 = 12 + 13 = 25
```

**示例 2：**

<img src="../../public/0129/num2tree.jpg" alt="num2tree.jpg"/>

```
输入: root = [4, 9, 0, 5, 1]
输出: 1026
解释: 
从根到叶子节点路径 4 -> 9 -> 5 代表数字 495
从根到叶子节点路径 4 -> 9 -> 1 代表数字 491
从根到叶子节点路径 4 -> 0 代表数字 40
因此, 数字总和 = 495 + 491 + 40 = 1026
```

<br/>

**提示：**

* 树中节点的数目在范围 `[1, 1000]` 内
* `0 <= Node.val <= 9`
* 树的深度不超过 `10`

---

### ❗ Solution（深度优先搜索）

<br/>

#### Java

```
/**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode() {}
 *     TreeNode(int val) { this.val = val; }
 *     TreeNode(int val, TreeNode left, TreeNode right) {
 *         this.val = val;
 *         this.left = left;
 *         this.right = right;
 *     }
 * }
 */
class Solution {
    public int sumNumbers(TreeNode root) {
        return dfs(root, 0);
    }

    private int dfs(TreeNode node, int prevSum) {
        // 如果当前节点为空, 表示不存在该路径, 返回 0
        if (node == null) {
            return 0;
        }

        // 如果当前节点不为空, 计算当前节点的数字之和 (将当前节点的数字添加到末尾)
        int currSum = prevSum * 10 + node.val;

        // 如果当前节点的左右子节点均为空, 即当前节点是叶子节点, 该路径搜索完毕, 返回当前节点的数字之和
        if (node.left == null && node.right == null) {
            return currSum;
        }

        // 如果当前节点是非叶子节点, 继续递归
        return dfs(node.left, currSum) + dfs(node.right, currSum);
    }
}
```

### ❓ Description

<br/>

对于字符串 `s` 和 `t`，只有在 `s = t + t + t + ... + t + t`（`t` 自身连接 `1` 次或多次）时，我们才认定 “`t` 能除尽 `s`”。

给定两个字符串 `str1` 和 `str2`。返回 *最长字符串 `x`，要求满足 `x` 能除尽 `str1` 且 `x` 能除尽 `str2`* 。

<br/>

**示例 1：**

```
输入: str1 = "ABCABC", str2 = "ABC"
输出: "ABC"
```

**示例 2：**

```
输入: str1 = "ABABAB", str2 = "ABAB"
输出: "AB"
```

**示例 3：**

```
输入: str1 = "LEET", str2 = "CODE"
输出: ""
```

**示例 4：**

```
输入: str1 = "AAAAAB", str2 = "AAA"
输出: ""
```

<br/>

**提示：**

* `1 <= str1.length, str2.length <= 1000`
* `str1` 和 `str2` 由大写英文字母组成

---

### ❗ Solution（最大公约数算法）

<br/>

#### idea

* 最大公约数算法：`gcd = (a, b) => ( b == 0 ? a : gcd(b, a % b) )`

<br/>

* 如果 `str1` 和 `str2` 有 **最大公因子** `str_gcd`
* 说明：
    * `str1` 由 `m` 个 `str_gcd` 组成
    * `str2` 由 `n` 个 `str_gcd` 组成

<br/>

* 将 `str1` 和 `str2` 根据不同顺序拼接成 `str1 + str2` 和 `str2 + str1` 两个字符串
* 由于 `str1` 和 `str2` 都是由多个 `str_gcd` 组成的
* 所以 `str1 + str2` 和 `str2 + str1` 两个拼接后的字符串是相同的，都是 `m + n` 个 `str_gcd`
* 如果不同，说明没有 **最大公因子**

<br/>

* 如果相同，说明有 **最大公因子**
* 且 **最大公因子的长度 = `str1.length()` 和 `str2.length()` 的最大公约数**
* 使用 `gcd` 算法求得 **两个字符串长度** 的 **最大公约数**
* 再根据长度截取，即可获得 **两个字符串的最大公因子**

<br/>

#### Java

```
class Solution {
    public String gcdOfStrings(String str1, String str2) {
        if (!str1.concat(str2).equals(str2.concat(str1))) {
            return "";
        }
        return str1.substring(0, gcd(str1.length(), str2.length()));
    }

    public int gcd(int a, int b) {
        int r = a % b;
        while (r != 0) {
            a = b;
            b = r;
            r = a % b;
        }
        return b;
    }
}
```

### â— Solution

<br/>

#### MySQL

```
SELECT c.name as customers
FROM customers AS c
LEFT JOIN orders AS o ON o.customerId = c.id
WHERE ISNULL(o.id);
```

<br/>

#### MS SQL Server

```
SELECT c.name as customers
FROM customers AS c
LEFT JOIN orders AS o ON o.customerId = c.id
WHERE o.id IS NULL;
```
